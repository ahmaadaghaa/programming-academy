أساسيات البرمجة

ابدأ رحلتك البرمجية بتعلم الأساسيات والمفاهيم الأساسية التي تحتاجها لأي مسار برمجي

لغة C

لغة برمجة قوية ومناسبة لفهم أساسيات البرمجة وهياكل البيانات

لغة C++

لغة برمجة كائنية التوجه مناسبة لتطوير التطبيقات المعقدة والأداء العالي

لغة C#

لغة برمجة متعددة الاستخدامات من مايكروسوفت لتطوير تطبيقات الويب والألعاب

قواعد البيانات SQL

لغة الاستعلامات البنيوية لإدارة وتنظيم البيانات في قواعد البيانات العلائقية

مسار Frontend Developer

تخصص في تطوير الواجهات الأمامية للمواقع والتطبيقات باستخدام أحدث التقنيات

HTML5

لغة ترميز النص الفائق لإنشاء هيكل صفحات الويب

CSS3

لغة تنسيق لتصميم وتخطيط صفحات الويب وجعلها جذابة بصريًا

JavaScript

لغة برمجة لتطوير الواجهات التفاعلية وتطبيقات الويب الديناميكية

React

مكتبة JavaScript لبناء واجهات المستخدم التفاعلية والقابلة لإعادة الاستخدام

Angular

إطار عمل قوي ومتكامل لبناء تطبيقات الويب الديناميكية

Vue.js

إطار عمل تقدمي لبناء واجهات المستخدم وتطبيقات الصفحة الواحدة

مسار Backend Developer

تخصص في تطوير الخوادم وقواعد البيانات والمنطق الخلفي للتطبيقات والمواقع

Node.js

بيئة تشغيل JavaScript من جانب الخادم لبناء تطبيقات الويب سريعة وقابلة للتطوير

Python & Django

لغة Python مع إطار عمل Django لبناء تطبيقات ويب قوية وآمنة

Java & Spring

لغة Java مع إطار عمل Spring لتطوير تطبيقات المؤسسات الكبيرة

خدمات السحابة

AWS, Azure, Google Cloud لنشر وإدارة التطبيقات في السحابة
